# Custom AI Chatbot with Llama 3

This is a personalized AI chatbot built using Python and powered by the Llama 3 API. It allows for seamless terminal-based conversations with one of the most advanced open-source LLMs.

## Features:
- **Llama 3 Integration:** High-quality AI responses.
- **Error Handling:** Robust communication with the API.
- **Secure Configuration:** API keys are managed separately for security.
